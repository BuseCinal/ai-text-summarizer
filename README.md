# AI Text Summarizer

This is a simple AI text summarization project built with Python.  
The program uses a pre-trained model from the Hugging Face **transformers** library to summarize long pieces of text.

The goal of this project is to practice **Natural Language Processing (NLP)** and understand how pre-trained deep learning models can be used in real applications.

## Features

- Summarizes long text automatically
- Uses a pre-trained NLP model
- Works with English and French text
- Simple command-line interaction
- Prevents summarizing very short texts
